# Campus Smart Planner

Campus Smart Planner is a simple Python program that helps students analyze their daily schedule and get basic study suggestions based on their wake-up time, school start time, and planned homework hours.

This project is designed for beginners and is suitable for high school coding competitions.

## Features

- Calculates morning free time
- Analyzes whether the schedule is tight or relaxed
- Evaluates daily homework planning
- Generates personalized study suggestions

## Requirements

- Python 3

No external libraries are required.

## How to Run

1. Download or copy the file `campus_smart_planner.py`.
2. Open a terminal or command prompt.
3. Run the program with:

python campus_smart_planner.py

4. Enter the required information when prompted.

## Example

Input:

Enter your wake-up time (e.g. 6:30): 6:30
Enter your school start time (e.g. 8:00): 8:00
Enter how many hours you plan to do homework today: 2

Output:

Daily Schedule Analysis
You have 90 minutes of free time in the morning.
You have plenty of time. You can read or study for 20 minutes.
You plan to do 2.0 hours of homework today.
Your homework plan is reasonable.

Today's Suggestion
Your schedule looks great. Keep going.

Good luck with your study.
